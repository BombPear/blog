# MarkDown 常用操作

一级标题：#+空格

二级标题：##+空格

三级标题：###+空格

粗体：**+文字+**    **中国**

斜体：*+文字+*     *中国*

删除线： ~ +文字+ ~  ~~中国~~

下划线：选中文本 ctrl + u

创建代码块：三个 ```

在同一个block内换行输入：shift+Enter

连续选择多个bolck：shift+鼠标选择或者方向键

设置block为黄色背景：/+Yellow Background

我是黄色背景

时间提醒：@+日期

May 1, 2021 9:00 AM 

有序列表： 数字 + . +空格        如：1.

1. 中国
2. 美国

无序列表： - + 空格

- 故宫
- 长城

To事项： [ + ]

- [ ]  洗衣服
- [x]  睡觉

展开清单： > + 空格       并且换行后摁Tab键，会自动展开此项

- 国家
- 省份
    1. 河南省
    2. 安徽省  

引用内容快：双引号+空格

> 时尚是一个轮回

创建分隔符：三个-
